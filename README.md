# Netflix-Data-Analytics-AbhaGarg-SBMC1-003
End-to-end data analytics project on Netflix Movies &amp; TV Shows using Python, SQL, and Power BI

## Project Overview
This project presents an end-to-end data analytics solution built on the Netflix Movies & TV Shows dataset.
The objective is to clean raw data, perform exploratory analysis, execute SQL-based business queries,
and visualize insights through an interactive Power BI dashboard.

## Dataset
- Source: Netflix Movies & TV Shows dataset
- Total Records: ~8,800 titles
- Attributes include content type, title, country, rating, genre, duration, release year, and date added.

## Tools & Technologies Used
- Python (Pandas, NumPy): Data cleaning and exploratory data analysis
- SQL (SQLite): Structured business queries
- Power BI: Interactive dashboard and visualization
- GitHub: Version control and project sharing

## Project Structure
data/ -> Raw and cleaned datasets, SQL database
python/ -> Jupyter notebook for analysis
powerbi/ -> Power BI dashboard file
report/ -> Final project report
presentation/ -> Presentation slides


## Key Analysis Performed
- Data cleaning and feature engineering
- Exploratory data analysis using Python
- SQL queries to answer business questions
- Interactive Power BI dashboard creation

## Key Insights
- Movies form ~70% of Netflix content, TV shows ~30%
- Content growth accelerated significantly after 2016
- Drama and Documentary genres dominate the catalog
- United States and India are major content contributors

## How to Run the Project
1. Open the Jupyter notebook in the `python/` folder to view analysis
2. Load `netflix_cleaned.csv` into Power BI to view the dashboard
3. Review the report and presentation for summarized insights
